# Imagens dos Objetivos do App

Para exibir as imagens dos objetivos na página inicial, coloque os arquivos de imagem nesta pasta com os seguintes nomes:

## Nomes dos arquivos esperados:

1. **objetivo-1.jpg** - Imagem para "Fortalecer a economia local"
   - Dimensões sugeridas: 800x600px ou proporção 4:3
   - Representa o fortalecimento da economia de Pirituba e entorno

2. **objetivo-2.jpg** - Imagem para "Conectar demanda e oferta"
   - Dimensões sugeridas: 800x600px ou proporção 4:3
   - Representa a facilitação do encontro entre demanda e oferta local

3. **objetivo-3.jpg** - Imagem para "Experiência digital moderna"
   - Dimensões sugeridas: 800x600px ou proporção 4:3
   - Representa a experiência digital moderna para o bairro

## Formatos suportados:
- JPG/JPEG
- PNG
- WebP
- AVIF

## Nota:
Se alguma imagem não for encontrada, será exibido um gradiente degradê azul-verde como placeholder.
