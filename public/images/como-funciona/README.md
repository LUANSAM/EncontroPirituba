# Imagens da Seção "Como Funciona"

Para exibir as imagens da seção "Como funciona" na página inicial, coloque os arquivos de imagem nesta pasta com os seguintes nomes:

## Nomes dos arquivos esperados:

1. **cliente-comum.jpg** - Imagem para "Cliente comum"
   - Dimensões sugeridas: 800x600px ou proporção 4:3
   - Representa acesso livre e gratuito para descobrir serviços locais

2. **profissionais.jpg** - Imagem para "Profissionais"
   - Dimensões sugeridas: 800x600px ou proporção 4:3
   - Representa plano por tokens para promover serviços e ampliar alcance

3. **estabelecimentos.jpg** - Imagem para "Estabelecimentos"
   - Dimensões sugeridas: 800x600px ou proporção 4:3
   - Representa tokens para campanhas, vouchers e mais visibilidade na região

## Formatos suportados:
- JPG/JPEG
- PNG
- WebP
- AVIF

## Nota:
Se alguma imagem não for encontrada, será exibido um gradiente degradê verde-laranja como placeholder.
